
# OCR-EKYC Demo
This project show OCR and EKYC frontend demo for InterITS


## Tech Stack
- React
- Tailwind CSS
- Vite

## How to use
1. Clone the repo
``` bash
git clone https://github.com/nguyenhoan1209/ocr_demo.git
```

2. Install dependencies
``` bash
npm install
```

3. Start the dev server
``` bash
npm run dev
```
